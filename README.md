# Sorting Visualizer

An interactive web-based Sorting Visualizer built using HTML, CSS, and JavaScript to help users understand how different sorting algorithms work through step-by-step animations and visual explanations.

## Features

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort
* Play / Pause / Previous / Next controls
* Speed adjustment slider
* Complexity explanation box
* Custom input mode for user-defined values
* Animated final sorted output
* Responsive and clean UI design

## Tech Stack

* HTML
* CSS
* JavaScript

## Project Structure

* `index.html` → Main sorting visualizer
* `custom-input-page.html` → Custom input mode
* `style.css` → Styling and UI design
* `script.js` → Sorting logic and visualization

## How to Run

1. Clone the repository
2. Open the project folder
3. Run `index.html` in your browser

No installation required.
